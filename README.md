### 🤖 CodeAlpha Basic Chatbot
A simple text-based chatbot built in Python as part of the CodeAlpha Python Internship. It simulates conversation using rule-based keyword matching rather than a full ML/NLP model.

**Features:**
* Predefined responses for greetings, farewells, and common questions
* Input normalization (lowercasing, stripping) for consistent matching
* Default fallback response for unrecognized input
* Runs in a loop until the user types an exit command like `bye` or `exit`

**Concepts Used:** Dictionaries · String methods (`.lower()`, `.strip()`) · `if-elif-else` control flow · `while` loops

**Tech Stack:** `Python 3` `VS Code` `CLI`

**How to Run:**
```bash
# 1. Install Python
https://www.python.org/downloads/

# 2. Clone the repository
git clone https://github.com/Minni-Zaid/codealpha-basic-chatbot.git

# 3. Run the chatbot
python chatbot.py
```

### 📊 CodeAlpha Stock Portfolio Tracker
A CLI-based Stock Portfolio Tracker built in Python as part of the CodeAlpha Python Internship. It calculates total investment value from user-entered stock details and predefined prices.

**Workflow:**
1. Takes the stock name from the user
2. Takes the quantity of shares
3. Looks up the price from a predefined stock price dictionary
4. Calculates and displays the total investment value

**Concepts Used:** Dictionaries · User input handling · Conditional statements · Arithmetic operations

**Tech Stack:** `Python 3` `VS Code` `CLI`

**How to Run:**
```bash
# 1. Install Python
https://www.python.org/downloads/

# 2. Clone the repository
git clone https://github.com/Minni-Zaid/codealpha-stock-portfolio-tracker.git
```
